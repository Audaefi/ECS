# ECS (E-Commerce Scraper)

**프로젝트 분야** : Cloud / Data Engineering

**프로젝트 기간** : 2022.02 ~ 2022.3 / Present (지속 개발 & 운용 중)

**프로젝트 설명** : Marketplace Research를 위한, AWS 기반 데이터 파이프라인 [수집-저장-처리-분석-시각화] 구축
